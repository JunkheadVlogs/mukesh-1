# Universal Safe Protection Guidelines

IMPORTANT — PRESERVE CURRENT WEBSITE EXACTLY AS IT IS.

Do NOT change, redesign, regenerate, replace, resize, reposition, restyle, restructure, or modify ANY visual or frontend element of the website.

The current live UI/UX, layout, and luxury boutique aesthetic must remain 100% identical after applying changes.

STRICTLY DO NOT MODIFY:
- overall layout
- mobile layout
- desktop layout
- responsive behavior
- hero section
- hero image
- image quality
- image positions
- image crop
- image dimensions
- category cards
- typography
- text positions
- buttons
- section spacing
- padding or margins
- transparent navbar
- overlays
- gradients
- blur effects
- colors
- Tailwind classes
- CSS styling
- component structure
- z-index layers
- animations
- transitions
- scroll behavior
- hover effects
- sliders/carousels
- product cards
- logo size/alignment
- icon positions
- header/footer design
- loading visuals

Do NOT:
- regenerate frontend
- auto-redesign sections
- modernize UI
- change theme
- optimize visually
- adjust spacing automatically
- alter responsive breakpoints
- replace existing styling

ONLY perform requested backend/performance/SEO optimization while keeping frontend appearance pixel-perfect identical.

Requirements:
- zero layout shift
- zero UI flicker
- zero hydration mismatch
- zero spacing changes
- zero responsive changes
- zero image movement
- zero text movement

Before deployment:
1. Create checkpoint backup
2. Generate preview deployment first
3. Compare preview with current live website
4. Automatically revert if ANY visual change is detected
5. Deploy ONLY if website appearance remains exactly the same
